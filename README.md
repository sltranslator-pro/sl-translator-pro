# SL Translator Pro

**AI-Powered Chat Translation for Second Life / Firestorm Viewer**

Automatically translate your outgoing chat and IM messages in real-time.

## Features

- **Local Chat Translation** - Your messages are automatically translated before sending
- **IM Translation** - Private messages are also translated
- **17+ Languages** - English, German, French, Spanish, Italian, Portuguese, Russian, Japanese, Chinese, Korean, Arabic, Dutch, Polish, Swedish, Turkish, Czech, Romanian
- **On-the-fly Language Switching** - Change target language without restarting
- **Firestorm Integration** - Automatic incoming translation setup
- **License System** - HWID-based activation

## Installation

1. Download **SLTranslatorPro_Setup.exe** from [Releases](https://github.com/sltranslator-pro/sl-translator-pro/releases)
2. Run the installer (auto-detects Firestorm)
3. Start Firestorm
4. Enter your license key and select your language
5. Start chatting!

## Usage

### Chat Commands
| Command | Description |
|---------|-------------|
| `/tr` | Show current status |
| `/tr en` | Set target language to English |
| `/tr de` | Set target language to German |
| `/tr fr` | Set target language to French |
| `/tr off` | Disable translation |
| `/tr on` | Enable translation |

### Keyboard Shortcut
- **Ctrl+Shift+T** - Open settings window

## How It Works

```
You write in your language
    → SL Translator Pro translates to target language
        → Other users receive your message in their language

Other users write in their language
    → Firestorm auto-translates to your language (built-in)
```
